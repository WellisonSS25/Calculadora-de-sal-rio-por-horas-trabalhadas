# Calculadora de Salário por Horas Trabalhadas

## Descrição

Este programa calcula o salário de um funcionário com base na quantidade de horas trabalhadas e no valor recebido por hora.  
Os dados são lidos via entrada padrão, o salário é calculado pela multiplicação das horas trabalhadas pelo valor da hora e o resultado é exibido seguindo rigorosamente o formato exigido.

## Entrada

O programa recebe, nesta ordem:
- Número do funcionário (inteiro)
- Quantidade de horas trabalhadas (inteiro)
- Valor recebido por hora (número decimal com duas casas decimais)

## Exemplo de Execução

### Entrada
25
160
15.50

### Saída Esperada
FUNCIONARIO = 25
SALARIO = R$ 2480.00
